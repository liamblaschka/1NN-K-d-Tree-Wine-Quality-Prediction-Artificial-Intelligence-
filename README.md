# 1NN (K-d Tree) Wine Quality Prediction (Artificial Intelligence)
Artificial Intelligence Assignment

## Description
In this assignment, we provide features obtained from physicochemical statistics for each white wine sample and its corresponding rating provided by wine experts. You are expected to implement the k-d tree (KDT) and use the training set to train your k-d tree, then provide wine quality prediction on the test set by searching the tree.


## Overview
This project predicts wine quality using a k-Nearest Neighbour model on structured data, supported by a k-d tree for efficient nearest neighbour search.

## Approach
- Implemented:
  - k-d tree construction using median splitting
  - 1-Nearest Neighbour search
- Used physicochemical features from the wine dataset
- Focused on efficient search using tree-based structure

## Key Learnings
- k-d trees enable efficient nearest neighbour search by reducing the number of comparisons required
- Nearest neighbour methods are simple but effective for structured data
